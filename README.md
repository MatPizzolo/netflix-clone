## Primeros Pasos
### Instalación
1. Clona el repositorio
   ```sh
   git clone https://github.com/MatPizzolo/netflix-clone.git
   ```
2. Instala los paquetes NPM
   ```sh
   npm install
   ```
3. Ejecuta la aplicación
   ```sh
   npm run start
   ```
