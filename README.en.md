## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MatPizzolo/netflix-clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app
   ```sh
   npm run start
   ```
